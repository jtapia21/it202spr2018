Answers to questions for Homework 4

  1. The Box Model is used when talking about design and layout which consist of margins, borders, padding, and the content.
  2. A block level element occupies the space of the parent element.
  3. The div tag is a block level element it will start in a new line.
  4. Inline elements are those which only occupy the space bounded by the tags defining the element, instead of breaking the flow of the content.
  5. The span tag is an inline level elementit will not start on a new line.
  6. We can add internal style sheet and inline styles in out pages.
  7. A CSS selector is the part of a CSS rule set that actually selects the content you want to style.
  8. body { background-color: #0000FF; }
